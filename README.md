# Bahasa Assembler Demo

## Instalasi NASM
Petunjuk instalasi NASM untuk berbagai sistem operasi.

## Program
- **Penjumlahan**: `sum.asm`
- **Perkalian**: `mul.asm`
- **Pengurangan**: `sub.asm`
- **Pembagian**: `div.asm`
- **Loop**: `loop.asm`
- **Stack dan Pop**: `stack_pop.asm`
- **Conditional**: `conditional.asm`
